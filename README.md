> Simple Todo list app built with React hooks and TypeScript.

### 🏠 [主页](https://github.com/liancheng-zcy/todoList_lac#readme)

## 环境
- node >=11.0.0
- npm >=6.0
- yarn >=1.16.0

## 安装

```sh
npm install
yarn
```

## 使用

```sh
npm run start
yarn start
```

## 运行 test

```sh
npm run test
```
